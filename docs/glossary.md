Daftar istilah penting dalam perjalanan Fullstack AI Engineer  
(Key terms in the journey to becoming a Fullstack AI Engineer)

## Fondasi Teknis / Technical Foundation
- **Git**: Sistem kontrol versi untuk melacak perubahan kode.  
  *Version control system to track code changes.*
- **Python**: Bahasa pemrograman utama untuk AI/ML.  
  *Primary programming language for AI/ML.*
- **Virtual Environment**: Lingkungan terisolasi untuk dependency.  
  *Isolated environment for dependencies.*

## Core AI & ML
- **Regression**: Model prediksi nilai kontinu.  
  *Model for predicting continuous values.*
- **Classification**: Model prediksi kategori.  
  *Model for predicting categories.*
- **Clustering**: Teknik pengelompokan data tanpa label.  
  *Technique for grouping unlabeled data.*

## MLOps & Data Engineering
- **Docker**: Platform untuk containerisasi aplikasi.  
  *Platform for containerizing applications.*
- **CI/CD**: Continuous Integration / Continuous Deployment.  
  *Automated integration and deployment pipeline.*
- **Pipeline**: Alur otomatis untuk data dan model.  
  *Automated workflow for data and models.*

## Fullstack Integration
- **REST API**: Antarmuka untuk komunikasi antar aplikasi.  
  *Interface for communication between applications.*
- **Frontend**: Bagian aplikasi yang berinteraksi dengan user.  
  *Part of the application that interacts with users.*
- **Backend**: Bagian aplikasi yang mengelola logika dan data.  
  *Part of the application that handles logic and data.*

## Freelance Readiness
- **Portfolio**: Kumpulan proyek untuk showcase ke klien.  
  *Collection of projects to showcase to clients.*
- **Proposal**: Dokumen penawaran kerja ke calon klien.  
  *Work offer document for potential clients.*
